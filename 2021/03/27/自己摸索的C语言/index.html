指针

其实学起来不难会用才叫难。

指针：其值为另一个变量的地址。

指针变量：就是变量存储了一分数据的指针。

例如：现在假设有一个 char 类型的变量 c，它存储了字符 ‘K’（ASCII码为十进制数 75），并占用了地址为 0X11A 的内存（地址通常用十六进制表示）。另外有一个指针变量 p，它的值为 0X11A，正好等于变量 c 的地址，这种情况我们就称 p 指向了 c，或者说 p 是指向变量 c 的指针。





由于指针储存的内容是地址所以给指针赋值的时候需要用上取地址符&，例如：int *p=&a;

在定义指针的时候必须带有*，但是再次给指针赋值的时候不带*，例如：int *p; p=&b;

CPU读写数据必须要知道数据在内存中的地址，普通变量和指针变量不过是地址的助记符，虽然p和a获得的数据一样,但是他们运行过程稍有不同，a一次就可以获取数据，p需要两次。

例如变量a、p地址为0X1000、0XF0A0，指向关系如下：


也就是说，使用指针是间接获取数据，使用变量名是直接获取数据，前者比后者的代价要高。

一个小小的案例:

   #include <stdio.h>

int main ()
{
   int  var = 20;   /* 实际变量的声明 */
   int  *ip;        /* 指针变量的声明 */

   ip = &var;  /* 在指针变量中存储 var 的地址 */

   printf("Address of var variable: %p\n", &var  );

   /* 在指针变量中存储的地址 */
   printf("Address stored in ip variable: %p\n", ip );

   /* 使用指针访问值 */
   printf("Value of *ip variable: %d\n", *ip );

   return 0;
}

执行完会有

Address of var variable: bffd8b3c
Address stored in ip variable: bffd8b3c
Value of *ip variable: 20

很明显在 ip与*ip有所不同，ip取的是地址，*ip取的是值

指向指针的指针

指向指针的指针是一种多级间接寻址的形式，或者说是一个指针链。通常，一个指针包含一个变量的地址。当我们定义一个指向指针的指针时，第一个指针包含了第二个指针的地址，第二个指针指向包含实际值的位置。

当一个目标值被一个指针间接指向到另一个指针时，访问这个值需要使用两个星号运算符，如下面实例所示：

int main ()
{
       int  var;
       int  *ptr;
       int  **pptr;

       var = 3000;

       /* 获取 var 的地址 */
       ptr = &var;

       /* 使用运算符 & 获取 ptr 的地址 */
       pptr = &ptr;

       /* 使用 pptr 获取值 */
       printf("Value of var = %d\n", var );
       printf("Value available at *ptr = %d\n", *ptr );
       printf("Value available at **pptr = %d\n", **pptr);

       return 0;
}

会输出

Value of var = 3000
Value available at *ptr = 3000
Value available at **pptr = 3000

传递指针给函数

例如：

#include <stdio.h>
#include <time.h>

void getSeconds(unsigned long *par);

int main ()

       printf("Number of seconds: %ld\n", sec);

       return 0;
}

void getSeconds(unsigned long *par)
{
       /* 获取当前的秒数 */
       *par = time( NULL );

C结构体

结构是 C 编程中另一种用户自定义的可用的数据类型，它允许您存储不同类型的数据项。

定义结构，必须用struct语句。struct语句定义了一个包含多个成员的新的数据类型
格式如下：

struct tag { 
    member-list
    member-list 
    member-list  
    ...
} variable-list ;

Tag 是结构体标签。

member-list 是标准的变量定义，比如 int i; 或者 float f，或者其他有效的变量定义。

variable-list 结构变量，定义在结构的末尾，最后一个分号之前，您可以指定一个或多个结构变量。

在一般情况下，tag、member-list、variable-list 这 3 部分至少要出现 2 个。

初始化结构体变量：

struct Books
{
       char  title[50];
       char  author[50];
       char  subject[100];
       int   book_id;
} book = {"C 语言", "RUNOOB", "编程语言", 123456}; //与上面相互对应

int main()
{
    printf("title : %s\nauthor: %s\nsubject: %s\nbook_id: %d\n", book.title, book.author, book.subject, book.book_id);
}

会输出

title : C 语言
author: RUNOOB
subject: 编程语言
book_id: 123456

我们也可以不必在struct的末尾定义一个变量 例如：

#include <stdlib.h>
#include <stdio.h>
#include<string.h>
struct Books
{
   char  title[50];
   char  author[50];
   char  subject[100];
   int   book_id;
};                                        //千万别忘记加逗号

int main()
{
    struct Books book1;                    //定义变量
    struct Books book2;
    strcpy (book1.title,"I love Java"); //<string.h>提供的为变量赋字符串的功能
    strcpy (book1.author,"ZWZ");
    strcpy (book1.subject,"JAVA");
    book1.book_id = 1000;

    printf("Title =%s\n",book1.title); //由于是字符串，要用%s
    printf("Author =%s\n",book1.author);
    printf("Subject =%s\n",book1.subject);
    printf("id =%d\n",book1.book_id);
    system("pause");
    return 0;
}

也会输出

Title =I love Java
Author =ZWZ
Subject =JAVA
id =1000

共用体

共用体是一种特殊的数据类型，允许您在相同的内存位置存储不同的数据类型。您可以定义一个带有多成员的共用体，但是任何时候只能有一个成员带有值。共用体提供了一种使用相同的内存位置的有效方式

基本格式是：

union union tap
{    
    标准的变量定义
    标准的变量定义
    ...
}[one or more union variables]

在使用共同体时候应注意不能一次性给多个变量赋值例如：

#include <stdio.h>
#include <string.h>

union Data
{
   int i;
   float f;
   char  str[20];
};

int main( )
{
   union Data data;        

   data.i = 10;
   data.f = 220.5;
   strcpy( data.str, "C Programming");

   printf( "data.i : %d\n", data.i);
   printf( "data.f : %f\n", data.f);
   printf( "data.str : %s\n", data.str);

   return 0;
}

会出现 

data.i : 1917853763
data.f : 4122360580327794860452759994368.000000
data.str : C Programming

i和f的值均有损耗没事因为最后付给变量的值占用了内存位置，即str成员可以完好输出的原因。但是换种方法就可以输出完全例如：

#include <stdio.h>
#include <string.h>

union Data
{
   int i;
   float f;
   char  str[20];
};

int main( )
{
   union Data data;        

   data.i = 10;
   printf( "data.i : %d\n", data.i);

   data.f = 220.5;
   printf( "data.f : %f\n", data.f);

   strcpy( data.str, "C Programming");
   printf( "data.str : %s\n", data.str);

   return 0;
}

会输出

data.i : 10
data.f : 220.500000
data.str : C Programming

位域

基本的定义语法是 

struct /* 冒号后面表示可识别的2进制位数 */
{
  unsigned int widthValidated ： ;
  unsigned int heightValidated;
} status ： ;

例子：

#include <stdio.h>
#include <string.h>

struct
{
  unsigned int age : 3;
} Age;

int main( )
{
   Age.age = 4;
   printf( "Sizeof( Age ) : %d\n", sizeof(Age) );
   printf( "Age.age : %d\n", Age.age );

   Age.age = 7;
   printf( "Age.age : %d\n", Age.age );

   Age.age = 8; // 二进制表示为 1000 有四位，超出
   printf( "Age.age : %d\n", Age.age );

   return 0;
}

会输出

Sizeof( Age ) : 4
Age.age : 4
Age.age : 7
Age.age : 0

很明显，把10进制转化为二进制的时候超过了位域规定的位数，则会使数据直接损坏。

typedef 和 #define 区别

  1. typedef只可以为类型定义字符名称，#define不仅可以定义字符名称，还可以为数值定义名称 即#define True one 也是可以的
  2. typedef 由编译器执行解释，#define语句由预编译器进行处理的。

输入与输出

  1. int getchar(void)只会读取单一的字符。可以用循环的方法获取多个字符
  2. int putchar(int c)一次只会输出一个单一字符，亦可用循环获取。
  3. char * gets(char*s)函数是读取一行s到缓冲区，直到终止符或EOF；
  4. int puts(const char*s)函数吧字符串s和一个尾随的换行符写入stdout中。
#include <stdio.h>

int main( )
{
   char str[100];

   printf( "Enter a value :");
   gets( str );

   printf( "\nYou entered: ");
   puts( str );
   return 0;
}

会弹出 i love hxm

递归

递归指的是在函数的定义中使用函数自身的方法。一个简单的求阶乘的例子：

#include <stdio.h>

double factorial(unsigned int i)
{
   if(i <= 1)
   {
      return 1;
   }
   return i * factorial(i - 1);
}
int  main()
{
    int i = 15;
    printf("%d 的阶乘为 %f\n", i, factorial(i));
    return 0;
}
